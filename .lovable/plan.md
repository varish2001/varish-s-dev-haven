
## Portfolio Website for Mohd Varish

A modern, dark-themed portfolio with blue accents, smooth animations, and all requested sections.

### Design
- Dark background (#0F172A) with blue accents (#3B82F6)
- Inter font for body, monospace for code elements
- Subtle fade-in animations on scroll, hover effects on cards
- Fully responsive with mobile hamburger menu

### Pages & Structure
Since this is a portfolio with smooth scrolling between sections on one page, I'll build it as a single-page site with anchor navigation (appropriate here as the user explicitly wants smooth scrolling between sections).

### Components to Build
1. **Navbar** — Fixed top nav with smooth scroll links, mobile hamburger menu, scroll progress indicator
2. **Hero Section** — Name, title, tagline, CTA buttons, animated code-style background with floating brackets/symbols
3. **About Section** — Bio with key highlights (internships, LeetCode, interests) in a clean layout
4. **Skills Section** — Grid of skill cards grouped by category with hover effects
5. **Projects Section** — 3 project cards (WanderLust, Know Your Roots, To-Do List) with tech badges, GitHub/demo links, hover lift effects
6. **Experience Section** — Timeline UI for 3 internships (Microsoft AICTE, Microsoft & SAP, AWS Academy)
7. **Achievements Section** — Cards/badges for LeetCode, DSA course, hackathons
8. **Contact Section** — Contact info links + frontend-only contact form
9. **Footer** — Social links (Email, LinkedIn, GitHub)

### Features
- Scroll progress bar at top of page
- Fade-in animations as sections enter viewport
- Card hover effects with shadows and scale
- Loading animation on initial page load
- Smooth scrolling between sections
